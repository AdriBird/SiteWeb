Les booléens

Voici les exercices du chapitre fonctions et leur correction :

Exercice 20 :

Affichez la valeur de l’expression :

4 == 5

En Python, "4 == 5" est appelé une expression.

Correction:
Le résultat renvoyé est False.

Exercice 21 :

Soit le programme suivant :

a = 4
b = 7

Quel est le résultat attendu après l'exécution de ce programme si vous affichez la valeur de "a==b" ?

Correction:
Le résultat renvoyé est False.

Exercice 22 :

Soit le programme suivant :

a = 4
b = 7

Quel est le résultat attendu après l'exécution de ce programme si vous affichez la valeur de "a!=b" ?

Correction:
Le résultat renvoyé est True.

Exercice 23 :

Soit le programme suivant :

a = 4
b = 7

Quel est le résultat attendu après l'exécution de ce programme si vous affichez la valeur de "a < b" ?

Correction:
Le résultat renvoyé est True.
