Variables

Les données utilisées dans un programme sont stockées dans des variables.
Ceci se fait par une affectation qui associe une donnée, représentée par une valeur ou une expression, avec un nom.
On peut imaginer une boite sur laquelle est écrit un nom et dans laquelle on place des informations diverses.
Le nom peut être n’importe quelle chaine alphanumérique, excepté certains mots réservés, sans commencer par un chiffre.
L’affection se fait à l’aide de l’opérateur « = ».
Exemple : l’instruction « coucou=3 » affecte la valeur 3 à la valeur variable de nom coucou.

A retenir :

  • Une variable est composé d’un nom (ou identificateur), d’une adresse en mémoire où est enregistrée une valeur (ou un ensemble de valeurs), d’un type qui définit ses propriétés.
  • Une expression a une valeur qui est le résultat d’une combinaison d’opérations sur des variables.
  • Une instruction est une commande qui doit être exécutée par la machine.

Une expression se calcule, elle a une valeur alors qu’une instruction s’exécute et n’a pas de valeur.

  • Une affectation (en Python) est une instruction qui commande à la machine de créer une variable en lui précisant son nom et sa valeur.

Détaillons ce qui se passe dans l’Exercice 3:

  • nous créons une variable a qui référence l'entier 11
  • nous affichons à l'écran la valeur référencée par a (c'est-à-dire 11)

La suite est un peu plus complexe, mais très importante à comprendre. Il va falloir lire la ligne "a = a + 1" de droite à gauche, décortiquons cette ligne :

  • "a + 1" : nous prenons la valeur actuelle de a (c'est-à-dire 11) et nous ajoutons 1 à 11, à droite de l'égalité nous avons donc maintenant la valeur 12
  • nous attribuons la valeur qui vient d'être calculée à la variable a
  • nous affichons à l'écran la nouvelle valeur référencée par a
