Les fonctions

Voici les exercices du chapitre fonctions et leur correction :

Exercice 15 :
Testez le programme suivant :

def ma_fonction(x):
  y = 3 * x + 2
  return y
solution = ma_fonction(4)
print(solution)

Il faut savoir qu'au moment de l'exécution de votre programme le code ma_fonction(4) sera systématiquement remplacé par la valeur renvoyée par la fonction (toujours dans notre exemple le ma_fonction(4) sera remplacé par le nombre 14).

Exercice 16 :
Quel est le résultat renvoyé par la fonction ci-dessous si on affiche une_autre_fonction (5,3) ?

def une_autre_fonction(x, b):
  y = 3 * x + b
  return y

Les paramètres peuvent être des chaînes de caractères (ainsi que la valeur retournée)

Correction:
Le résultat renvoyé est 18.

Exercice 17 :
Quel est le résultat attendu après l'exécution du programme ci-dessous et la saisie dans la console de dit_bonjour("toto", 14) ?

def dit_bonjour(nom, age):
  phrase = "Bonjour " + nom + ", vous avez " + str(age) + " ans."
  return phrase

Attention : remarquez bien les guillemets autour du paramètre "toto" (c'est une chaîne de caractères)
Les paramètres ne sont pas obligatoires.

Correction:
Le résultat attendu est : Bonjour toto, vous avez 14 ans.

Exercice 18 :
Testez la fonction suivante :

def ma_fon():
  return "voici une fonction qui ne sert à rien"

Il faut aussi savoir qu'une fonction ne renvoie pas forcément de valeur (le mot clé return n'est pas obligatoire). Mais si elle ne renvoie pas de valeur, elle peut faire plein de choses, par exemple elle peut tout simplement afficher une chaîne de caractères à l'aide d'un "print". Sachez que dans certains langages, on utilise les termes méthode ou procédure pour qualifier une fonction "qui ne renvoie rien".

Exercice 19 :
Soit le programme suivant :

def dit_bonjour(nom, age):
  phrase = "Bonjour " + nom + ", vous avez " + str(age) + " ans."
  print(phrase)

Executez dit_bonjour("toto", 14))

Correction:
Cette fonction renvoit : Bonjour toto, vous avez 14 ans.
